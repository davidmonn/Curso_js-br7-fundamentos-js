02 - Hello World

Intro:

Aqui sera mostrado como colocar verificar nosso codigo, para podermos verificar algumas informacoes que nao sao mostradas aos usuarios, podemos apertar f12 e ir na aba de console, para verificar se nosso codigo funcionou corretamente.

alert("");

Com ele conseguimos fazer basicamente assim que o usuario entre em nossa pagina, apareca uma mensagem em seu navegador

Obs: O alert() pausa a execução da página até o usuário clicar em “OK”. Por isso, hoje ele é mais usado para testes e aprendizado, não tanto em produção.

console.log("");

O console.log() é usado para exibir informações no console do navegador, servindo como uma ferramenta de apoio para o programador verificar se o código está funcionando corretamente e acompanhar valores de variáveis durante a execução. Ele não aparece para o usuário final, apenas para quem está desenvolvendo o sistema. É muito utilizado para testes, depuração de erros e entendimento do comportamento do código.

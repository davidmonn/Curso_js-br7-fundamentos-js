01 Modulo JS - Introducao ao JavaScript

JavaScript é usado para tornar o site dinâmico, interativo e funcional, coisa que HTML e CSS sozinhos não conseguem fazer.

Utilizades JS:

  • Abrir e fechar menu hambúrguer
  • Criar animações que respondem ao clique
  • Validar formulários antes de enviar
  • Mostrar/ocultar elementos
  • Trocar imagens, textos ou cores dinamicamente
  • Criar sliders, carrosséis e efeitos visuais
  • Manipular dados, fazer requisições, etc.

Hoje em dia o JavaScript não é limitado apenas a sites; ele se tornou uma linguagem independente, usada também em servidores, aplicativos, jogos e muito mais.

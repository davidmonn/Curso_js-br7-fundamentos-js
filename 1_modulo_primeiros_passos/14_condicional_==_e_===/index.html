Aula Anterior - Condicional If/Else

Condicional == e ===

Intro:

É usado para verificar se um valor é igual ao outro, sem levar o tipo em consideração. Ou seja, ele compara apenas o valor.

==

Funciona de forma parecida com o ==, porém é mais preciso, pois verifica o valor e o tipo. Se o valor for number, deve ser number. Se for string, deve ser string.

===

Basicamente com igual a === mas sendo mais precisa com o tipo de valor recebido. Resumindo precisa ser identico ao recebido valor recebido no if se for number o valor tambem tera que ser number, se for string devera receber valor em string

Resumindo

Se usarmos == e o usuário informar 20 como string, o valor será aceito mesmo assim. Já com ===, o valor só será verdadeiro se for idêntico em valor e em tipo no if do JavaScript.

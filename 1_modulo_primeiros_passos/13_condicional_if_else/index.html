Condicional if/else

Intro:

Quando trabalhamos com a condicional if, estamos aplicando uma condição onde, se ela for atendida, será executado o resultado que definimos dentro do if.

If:

Caso a condição do if não seja atendida, o código cairá automaticamente no else, onde podemos definir o que deve acontecer nessa situação.

else:

Caso a condicional if nao seja atendida, basicamente caira automaticamente na condicional else, onde tambem podemos adicionar o que pode ocorrer ao cair nessa condicional.

Condicionais

Temos várias formas de usar condicionais. Uma delas é verificar se um valor é maior que outro, menor, igual, entre outras comparações. Isso nos dá uma grande diversidade de formas de aplicar condicionais no código.

Alguns simbolos bem usados sao:

  • > maior
  • < menor
  • = = igual
  • >= maior ou igual
  • <= menor ou igual
  • != diferente

02 - Inserindo Arquivos JS na Pagina

Intro:

Iniciaremos criando um arquivo JavaScript, para isso pode ser dado qualquer nome, desde que tenha como final .js. Para o navegador conseguir fazer a leitura e executar esse arquivo js, dentro do body, melhor sendo no final, usamos uma tag chamada script e fazemos com ela, parecido com CSS onde adicionamos o atributo: src="aqui ira o local onde esta o arquivo", semelheante ao CSS Externo.

Importante:

Qual diferenca entre a tag script estar no body ou no head ?

Quando o script está no head, ele é carregado antes do conteúdo da página. Isso pode causar problemas, porque se o JavaScript tiver erro ou demorar para carregar, o restante da página pode ficar travado ou carregar mais lentamente. Por isso, normalmente colocamos o script no final do body, para que todo o HTML carregue primeiro, e só depois o JavaScript seja executado. Isso evita erros e melhora o desempenho.

Switch

Intro:

Switch, é basicamente uma condicional onde temos multiplos resultados.

Explicacao

Usamos o switch, quando queremos basicamente selecionar algo com multiplas escolha, um exemplo seria profissao. Onde teremos 3 profissoes, fiscal (verde), policial (azul) e bombeiro (vermelho).

Como usar Switch

Ele e usado semelheante ao if onde usamos a condicional switch() { case 'fiscal': break; } poderemos compreender melhor com a imagem abaixo de como fazer:

break

Serve para parar e nao ir para as demais condicionais.

default

Usada no final de nossa condicional, para quando nem uma opcao desejada for selecionada, damos uma por exemplo uma mensagem de "Selecao escolhida inexistente".

Mais informacoes

Essa uma condicional que nao e tao usada, mas dependendo de casos pode ser muito util.

Aula Anterior - Condicional Dupla (Else If)

Desmistificando o If

Intro:

Aqui sera explicado novamente sob condicionais, para facilitar a compreensao.

Explicacao

Quando utilizamos por exemplo um if, estamos fazendo basicamente uma verificacao de verdadeiro ou falso para oque desejamos, ou sejam isso nada mais e do que uma verificacao de boolean.

Resumindo o if e basicamente uma condicao que ira retornar um verdadeiro ou falso.

Condicional em Variavel

Assim como usamos condições dentro de um if, também podemos usar uma condicional diretamente na atribuição de uma variável. A diferença é que, nesse formato, só conseguimos trabalhar com duas opções (verdadeiro ou falso), pois não é possível criar condicionais múltiplas como fazemos com else if.

Opcoes

Isso não vale apenas para condicionais dentro de um if ou para variáveis. Também podemos usar uma condicional diretamente em um console.log, ou dentro do próprio if, ou praticamente em qualquer lugar do código. Ou seja, podemos aplicar esse tipo de condição onde for necessário, desde que a lógica faça sentido.

Aula Anterior - Variaveis

02 - Tipos de Dados

Intro:

Para trabalharmos com JS precisamos tambem entender os tipos de dados que usaremos e quando usar cada tipo, primcipalmente se tratando de variaveis.

Tipos de dados:

Strings:

Usada para armazenar textos em uma variável. Os valores sempre ficam entre aspas.

Number:

Usada para armazenar números, tanto inteiros quanto decimais.

Boolean:

Usada para representar valores lógicos, sendo apenas verdadeiro (true) ou falso (false).

Arrays

Usado para armazenar multiplos valores, por exemplo nomes, numeros e etc. É uma variavel que consegue adicionar mais de um valor.

Object:

É semelhante ao array por também armazenar vários valores, porém, no objeto cada valor possui uma característica (chave) que o identifica, como nome, idade, hobby, etc. Isso permite organizar melhor as informações, deixando cada dado ligado a uma descrição, o que facilita o acesso e a leitura no programa. Ex:
let informacoesUser = { nome: "David", idade: 28, hobby: "Violao" }

Verificar tipo

Podemos tambem verificar o tipo de uma variavel com console.log(), para isso basta usarmos o typeof ficando da seguinte forma por exemplo:
console.log(typeof nome)

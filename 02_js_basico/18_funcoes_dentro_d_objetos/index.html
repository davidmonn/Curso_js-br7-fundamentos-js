Mais informacoes

Quando criamos uma funcao, normalmente a damos um nome, porem quando usamos em um objeto, ja demos um nome para ela, que no caso seria nesse exemplo usado o nomeCompleto.

this

Com o this, se refere ao proprio objeto ou seja, proprio item. Isso e basicamente o mesmo que termos usado a variavel pessoa (nome do objeto usado no exemplo), se no caso usassemos uma variavel normal como let, nao funcionaria pois como esta dentro de uma function, ela tem um escopo interno.

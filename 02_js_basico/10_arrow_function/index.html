Arrow function

Intro:

Aqui iremos ver algo considerado ainda como uma "novidade" do JS, mesmo um tempo apos ter sido implementado que sao as Arrow Functions.

Explicacao

Basicamente é uma forma diferente de definirmos uma funcao, ela ira funcionar e ser usada da mesma forma, mas diferenca de como ela sera criada.

Para que serve?

Com isso conseguimos criar funcoes com menos codigo.

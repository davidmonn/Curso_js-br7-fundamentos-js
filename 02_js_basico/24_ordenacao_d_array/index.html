Funcoes para ordenacao de array

sort()

A funcao sort() organiza os itens de um array em ordem alfabetica. E importante lembrar que ela altera o proprio array, mudando a ordem dos indices.

fruits.reverse()

Inverte toda a ordem do array. Pode ser usada junto com o sort(), criando assim uma lista em ordem alfabetica decrescente.

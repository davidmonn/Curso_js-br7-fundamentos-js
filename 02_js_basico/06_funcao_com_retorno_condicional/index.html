Funcao com Retorno Condicional

Intro:

Aqui iremos criar uma funcao que tera que ter um retorno condicional. Nessa funcao iremos basicamente ter que:

  • Receber idade
  • Dizer se é maior ou menor de idade

Importante lembrar

Um function pode ter apenas um return, porem, se houver uma condicional isso nao se aplica.

Loop While

Intro:

O while e semelheante ao for mas feito de forma diferente. Existem duas formas pois em alguns momentos sao mais necessarios fazer com while e outras com for. Na maioria das vezes ja iremos conseguir resolver o problema com o loop for, apenas algumas vezes iremos precisar do loop while

Codigo Usado de Exemplo

Oque e Loop While

While = Enquanto, ou seja, quando criamos uma condicao enquanto essa condicao for true ele ira executar determinado codigo. Importante lembrar que geralmente quando criarmos um while para nao termos problemas com repeticao infinita e travar nosso site, trocamos a variavel como por exemplo numero++

Quando usamos a variavel numero++; estamos dizendo que apos executar uma vez ira para 1, depois para 2, 3 e assim por diante

Criando While:

Uma das diferencas que podemos observar ao usarmos o while é que desmembramos o codigo, pois criamos uma variavel fora, depois colocamos a condicional (while) dentro da execucao alteramos o numero(numero++)

Criando com For:

Diferencas entre For e While

  • For -> usado para percorrer dados criados como em um array
  • While -> usado para criar ou preencher dados, quando basicamente nao sabemos exatamente quantas vezes vai repetir ou quando a repeticao depende so de uma condicao

Declarando uma Funcao

Intro:

Agora iremos ver como podemos estar criando uma funcao (function).

Como criar uma function

Existem várias formas de criar uma função no JavaScript. Aqui veremos a forma mais comum, usando uma declaração chamada function. Para isso faremos da seguinte forma:

  1. É a palavra-chave que usamos para iniciar a criação de uma função.
  2. Segundo: Damos um nome para que possamos chamá-la depois, como fazemos com variáveis. Podemos nomear usando camelCase (primeira palavra minúscula e as seguintes com inicial maiúscula).
  3. Após o nome, colocamos parênteses (), que futuramente podem receber parâmetros. Logo depois, abrimos e fechamos chaves {}, onde ficará o conteúdo da função.
  4. Dentro das chaves, escrevemos o que a função deve fazer, ou seja, o bloco de código que será executado quando chamarmos a função.

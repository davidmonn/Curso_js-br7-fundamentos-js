Iteracao de Array'

Intro:

Aqui iremos ver alguns metodos de interacao ou iteracao, dependendo de como preferir chamar, com o array. Usamos a iteracao para ter o array como referencia para podermos fazer algumas coisas.

Opcoes

Filter

Servindo para podermos filtrar alguma coisa. Exemplo: se tivermos um array de frutas, e queremos filtrar apenas as frutas que tem apenas mais de 4 letras. Como usa-lo: fruits.filter()

Importante: o filter ao ser usado, gera um novo array, entao iremos precisar criar uma variavel para poder salvar.

Filter Conclusa: com o filter, temos uma forma mais prática de percorrer um array. Nós criamos a regra (a função), e o filter fica encarregado de:

  • percorrer o array
  • aplicar essa regra em cada item
  • montar um novo array com os valores aprovados

Resumindo: O filter percorre o array automaticamente. Nós apenas definimos a condição que cada item precisa atender para entrar no array. O filter() pode receber até 3 parâmetros na função callback: value → o próprio item do array, index → a posição (índice) do item, array → o array completo (caso precisemos comparar)

every e some

every e some não são feitos para passar mensagens ao usuário, e sim para realizar verificações e validar condições. Ou seja, retornam apenas true (verdadeiro) ou false (falso).

Um exemplo seria verificar se as frutas possuem mais que 3 letras. Podemos usar o every (todos): se todas as frutas tiverem mais que 3 letras, o resultado será true. O every funciona de forma parecida com o operador lógico &&, onde todas as condições precisam ser verdadeiras para o resultado final ser true.

some(algum) para que ele possa dar true precisa que apenas algum item de nossa condicional seja verifida como true

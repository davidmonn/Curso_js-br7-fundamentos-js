Aula Anterior - Introducao a Array

Operacoes Basicas em Array

Intro:

Agora iremos ver algumas coisas basicas que podemos estar fazendo com array.

Exemplo usado:

length

Usado para podermos estar verificando a quantidade de itens dentro de um array. Para usarmos podemos fazer da seguinte forma:
console.log(`Total de ingredientes: ${ingredientes.length}`);

Adicionar mais itens

Se soubermos a quantidade de itens dentro de um array, podemos estar adicionando mais itens, utilizando o nome da variavel do array e o numero em que sera adicioanado, ficando da seguinte forma:
ingredientes[6] = 'Chocolate';

Se por acaso escolhermos o numero errado, aquele espaco ira pular para o proximo, ficando com o valor de empty (vazio).

Obs: essa e uma das formas menos utilizidas usada. Recomendado fazer de outra forma a adicao.

Adicionar mais Itens Forma Certa (push):

Para podermos adicionar mais itens dentro de nosso array podemos fazer da seguinte forma:
ingredientes.push('Chocolate')

Essa e a maneira recomendada, utilizando da funcao push

funcao pop

Usada para basicamente remover o ultimo item de nosso array

funcao shift

Ao contrario da funcao pop, usado para remover o primeiro item de nossa lista.

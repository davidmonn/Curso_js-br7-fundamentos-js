Passando Parametros em Funcoes

Intro:

Nesta aula vamos entender um pouco mais sobre parâmetros. Parâmetros são valores que passamos dentro dos parênteses () de uma função para que ela possa trabalhar com esses valores.

Explicacao

Para essa explicacao, usaremos de exemplo como se fossemos fazer uma function para somar, para isso tambem usaremos o parametro, dentro do parametro colocamos os dados que seram recebidos e utilizados dentro de nossa functiom. Quando criamos um parâmetro dentro de uma função, o valor desse parâmetro só existe dentro do bloco da função. Ou seja, ele só pode ser usado ali dentro, e não fora da função.

Como usar:

Apos termos feito nossa function e passado os parametros desejados, para usa-lo e bem simples. Basta invocar nossa function e passar os valores desejados dentro de nossa function, como utlizamos n1,n2 dentro de nosso parametro, iremos precisar passar dos valores dentro de nossa function somar.

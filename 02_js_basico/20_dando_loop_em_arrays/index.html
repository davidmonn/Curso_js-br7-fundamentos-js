Dando Loop em Arrays

Intro:

Agora iremos ver como dar loop em um array, atualmente utilizamos muito o loop quando estamos criando sistemas, muitas vezes o usando em arrays.

length com for

Algo muito util de estarmos usando, para caso de criarmos uma lista que estara em constante atualizacao, podemos estar utilizando o length, para nao precisarmos passar um valor fixo e dar como undefined. Ficando basicamente da seguinte forma:
for(let n = 0; n < cores.length; n++){}

Imagem de Exemplo - Usando For com Length

For

O for no JS tem diferentes formas de ser usado, essa que foi mostrada foi apenas uma das formas. Temos uma forma mais facil quando queremos fazer loop em um array, da seguinte forma:
for(let i in cores){
console.log(cores[i]);
}

Explicacao let i in

  • let i: estamos basicamente criando a variavel i.
  • in: significa para cada indice dentro de nossa array no caso cores.
  • cores[i]: acessa o valor usando o indice.

Let in I com function

Outra forma com mesmo resultado:

Ficando da seguinte forma:
for(let cor for cores) {
console.log(cor); }

Loop Letras Maiusculas

Com loops podemos percorrer dados e tratar valores, como converter textos para letras maiúsculas ou minúsculas.

Diferenca For in e For of:

  • for in: percorre indices (Índice → posição do elemento no array. Ex: 0, 1, 2...).
  • for of: percorre valores (Valor → conteúdo armazenado nessa posição. Ex: string, número, objeto, etc.).

IMPORTANTE:

Quando usamos for (let cor of cores), a variável cor recebe o valor do array. Como o array cores possui objetos, ao usar cor devemos tratá-lo como um objeto, por isso acessamos suas propriedades com cor.nome .

Já quando usamos for (let i in cores), a variável i representa o índice do array. Nesse caso, o valor ainda está dentro do array, então precisamos acessá-lo usando cores[i], ficando cores[i].nome.

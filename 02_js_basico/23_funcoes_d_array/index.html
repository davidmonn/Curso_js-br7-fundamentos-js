Fucoes de Array

Intro:

Quando estamos lidando com arrays, temos que aprender muitas coisas uteis e aqui iremos ver algumas funcoes basicas, que podemos executar dentro do proprio array.

Exemplo

Explicacao

  • console.log
    • Se usarmos apenas o console.log com a array ele nos mostra o array.
  • length
    • Agora se usamos por exemplo console.log(fruits.length) nesse exemplo mostrado, ele nos trara o resultado sendo 4, significando que temos 4 itens em nossa lista array.
  • push()
    • Adiciona mais itens ao nosso array, devemos lembrar que ele nao exclui os itens anteriores, apenas adiciona mais um. Ex de uso: console.log(fruits.push('manga')) com isso agora ele nos retornara 5.
  • pop()
    • Deleta o ultimo item de nosso array,
  • shift()
    • Deleta o primeiro item de nosso array
  • join()
    • Serve basicamente para juntar, pega um array converte para string e separa da forma que deixamos em nosso join. Ex: console.log(fruits.join(' ')), nesse caso dentro de nossa aspas, colocamos um espaco com isso ele ira separar com espaco.
  • Alterar item da lista
    • Podemos tambem alterar um item da lista. Ex se fizemos da seguinte forma fruits[0] = 'Pera' com isso nossa lista nao ira mais comecar com Maca mas com Pera. Ou seja, substituimos a Maca pela Pera.
  • fruits [fruits.length - 1] = 'Kiwi'
    • Dessa forma tambem substituimos nossa lista mas nesse caso sera alterado laranja para pera, pois seria o ultimo item -1.

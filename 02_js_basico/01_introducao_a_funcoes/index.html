Introducao a Funções

Intro:

Uma função (function) é um bloco de código criado para realizar uma tarefa específica e que podemos reutilizar sempre que precisarmos no programa. Podemos usar funções para diversas ações, como buscar nomes, fazer cálculos, formatar textos (por exemplo, deixar apenas a primeira letra maiúscula) e muito mais.

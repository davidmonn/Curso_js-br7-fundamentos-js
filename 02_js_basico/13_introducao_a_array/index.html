Introducao a Array

Intro:

Aqui iremos comecar a entender um array. Muito util e usada para podermos armazenar multiplos valores dentro de uma unica "Variavel", no caso uma array.

Explicacao

Nela podemos salvar coisas como por exemplo, uma lista de cores, produtos, nomes e etc.

Mais informacoes

Quando queremos criar apenas um valor, podemos apenas armazena-lo em uma variavel. Agora quando temos uma lista para armazenar, iremos precisar de um array.

Criando Array:

Para criarmos um array, e parecido com uma variavel, porem, nele usamos [''] e separados por virgula. Ex: let colors = ['Azul', 'Vermelho', 'Branco', 'Preto', 'Cinza', 'Green'];

Exibindo um Array

Temos algumas maneiras de exibir uma array, pode ser exibido tudo de uma vez, ex: console.log(colors); ou tambem podemos selecionar apenas um dos valores da seguinte forma por exemplo: console.log(colors[1]); Nesse caso, como usamos o valor 1, ele ira selecionar o vermelho, pois todo array, comeca a contagem pelo 0

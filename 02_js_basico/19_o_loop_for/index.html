O Loop For

Intro:

Loop basicamente em todas as linguagens de programacao e quando queremos que um codigo seja executado diversas vezes.

Exemplo

Quando criamos uma lista de produtos com um array, contendo cada produto, onde nele ira ter suas informacoes basicas como: preco, quantidade, validade e etc. Se quisermos exibir na tela as caracteristicas de todos os produtos, iremos precisar criar um loop,

Como Fazer Loop

Temos varias formas que podem ser feitas, uma delas e usando o for.

Loop For

É basicamente usado para percorrer cada posicao do array do inicio ao fim, executando uma acao para cada item

Criand For:

Muito semelheante ao criarmos um if. No for temos que colocar 3 coisas inicio do contador, condicional (para que o codigo continue executando) e por ultimo o n++ e a variavel + 1, pois seguira a ordem basicamente da seguinte forma: variavel iniciara em 0, enquanto a variavel for menor que 10 ela ira continuar executando e por ultimo apos ira acrescentando n + 1 Ficando da seguinte forma:
for(let n = 0; n < 10; n++){

} :

Resumindo No for precisamos informar três partes:

  • Início do contador
  • Condição (enquanto for verdadeira, o código continua executando)
  • Incremento (n++, que soma +1 a cada volta).

A ordem de execução é:

  • A variável começa em 0
  • Enquanto n < 10, o bloco será executado
  • Ao final de cada execução, n recebe +1

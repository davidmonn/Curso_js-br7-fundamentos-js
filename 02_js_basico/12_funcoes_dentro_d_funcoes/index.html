Function dentro de Function

Com essa mesmo function feita, podemos simplificar nosso codigo colocando uma function dentro da outra para podermos organizar melhor nosso codigo.

Quando usar:

  • A função interna só será usada ali
  • Você quer separar uma lógica menor dentro de uma maior
  • Ajudar na legibilidade e manutenção do código

Vantagens:

  • Código mais organizado
  • Menos nomes no escopo global
  • Mais controle sobre o que pode ou não ser acessado

Quando nao usar:

  • A função precisa ser usada em vários lugares
  • A lógica é genérica ou reutilizável
  • Deixar funções muito profundas (uma dentro da outra da outra…)

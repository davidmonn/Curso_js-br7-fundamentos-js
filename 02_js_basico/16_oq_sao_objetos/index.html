Oque sao Objetos

Intro:

Agora iremos ver um pouco sobre objetos. Eles são semelhantes a arrays, porém possuem outro propósito.

Uso

Assim como um array, os objetos permitem armazenar múltiplas informações em uma única variável. A diferença é que eles são criados de outra forma e utilizados com objetivos diferentes. Um objeto serve para representar algo com características. Por exemplo, uma pessoa possui nome, profissão, idade, etc. Nesse caso, “pessoa” seria o objeto principal, contendo essas informações.

Objeto Dentro de Objeto

Assim como podemos criar um array dentro de outro array, também podemos criar um objeto dentro de outro objeto. Por exemplo, podemos criar um personagem de jogo com características básicas, como nome e profissão. Dentro desse mesmo objeto, podemos ter outro objeto responsável pelos atributos do personagem, como nível, magia, stamina e força.

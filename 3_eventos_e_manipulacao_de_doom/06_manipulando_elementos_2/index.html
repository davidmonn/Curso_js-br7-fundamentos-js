Intro:

Agora iremos ver mais alguma maneiras de estarmos manipulando elementos.

Explicacao

Quando selecionamos um elemento, como no caso da aula passada onde selecionamos uma ul. Temos acesso a diversas propriedades desse elemento como tambem a metodos.

append()

Permite adicionar conteúdo ao final de um elemento, sem apagar o conteúdo que já existe. Ou seja, ele adiciona o novo conteúdo junto com o conteúdo atual. Porem so funciona para texto, nao funciona tags HTML

Diferenca de innerHTML e append

Quando usamos innerHTML (assim como innerText e textContent), o valor atribuído substitui todo o conteúdo existente do elemento. Ou seja, o conteúdo antigo é removido e substituído pelo novo conteúdo que você definiu.

Resumindo:

  • innerHTML = troca tudo
  • append = soma conteúdo

Utilidade Real:

Se estivermos trabalhando com uma lista muito grande por exemplo 1000 itens, onde desejamos apenas adicionar mais um item ao final, se fizemos isso com innerHTML ele ira pegar esse unico item que iremos adicionar, pegar os outros 1000, remover da memoria e vai re-adionar todos os itens novamente com 1 unico novo. Com isso podemos ver diferenca no processamento.

appendChild

O appendChild adiciona um elemento criado no JavaScript ao final de outro elemento no HTML. Antes de usar, é necessário criar o elemento com createElement e definir seu conteúdo.

prepend

O prepend adiciona conteúdo ou elementos no início de um elemento, sem apagar o conteúdo existente. É o contrário do append, que adiciona no final.

Metodos

Sao funcoes que temos no proprio elemento e servem para podermos manipulacoes nesse elemento.

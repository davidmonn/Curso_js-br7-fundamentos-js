Selecionando Elementos 02

Intro:

Aqui iremos focar em como selecionar elementos. Isso é essencial em grandes projetos, onde podem existir muitas divs e elementos iguais.

document.getElementById("")

Seleciona um elemento pelo ID. Muito útil, pois só podemos ter um único ID na página, facilitando a localização e modificação. Ex: document.getElementById("teste").

document.getElementsByTagName("")

Seleciona elementos pelo nome da tag. Ex: document.getElementsByTagName("div"). Pode retornar muitos elementos, então é menos prático em projetos grandes.

document.getElementsByClassName("")

Seleciona elementos pela class. Ex: document.getElementsByClassName("teste").

document.querySelector("")

Seleciona elementos usando a mesma sintaxe do CSS.
Ex: ID: querySelector("#teste"), Class: querySelector(".teste"), Tag: querySelector("h1").

Também podemos selecionar em cascata, por exemplo: ul > li.
document.querySelector("li") seleciona o primeiro li encontrado.
querySelectorAll("li") seleciona todos os li.
document.querySelectorAll("#teste ul li") seleciona todos os li dentro dessa estrutura.

Observacoes Importantes:

Elements

Sempre que usamos elements no plural, o retorno é uma HTMLCollection (que lembra muito um array). Mesmo que exista apenas um elemento, ele retorna como lista.

elementById("")

Quando usamos getElementById, ele retorna apenas um elemento, pois ID é único e não pode se repetir.

elementsByClassName("")

Aqui usamos elements (plural), pois podemos ter vários elementos com a mesma class, então ele retorna uma lista.

querySelectorAll("")

Sempre retorna uma NodeList (que lembra um array), mesmo se existir apenas um elemento selecionado.

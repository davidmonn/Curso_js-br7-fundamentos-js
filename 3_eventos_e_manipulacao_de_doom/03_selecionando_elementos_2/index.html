Selecionando Elementos 02

Intro:

Aqui iremos focar em como selcionar elementos, isso e essencial quando estamos trabalhando com grandes projetos onde pode ter diversas divs iguais.

document.getElementById("")

Div com ID, muito util pois so podemos ter um item com o nome da id, facilitando na localizacao para modificacao. Ex: document.getELementById("teste").

document.getElementsByTagName("")

Usado para poder selecionar a tag pelo nome, exemplo: document.getElementsByTagName("DIV"). Menos eficiente em grandes Projetos.

document.getElementsByClassName("")

Usado para podermos fazer a selecao por meio da class. Ex: document.getElementsClassName("").

document.querySelector("")

Com ele conseguimos selecionar basicamente como um CSS, ex: id: querySelector("#teste"), class: querySelector(".teste"), tag: querySelector("h1"). Com ele podemos ate fazerr efeito casacata, onde se por exemplo houver um elemento dentro de outro como: ul > li, dessa forma em cascata, poderiamos fazer da seguinte forma para selecionar: document.querySelector("li") ou se quiser todos querySelectorAll("li"), para efeito camada temos document.querySelectorAll("#teste ul li"), dessa forma se tivermos uma ul com duas li diferentes, conseguimos selecionar a primeira li.

Observacoes Importantes:

Elements

Sempre que usamos elements no plural, temos que nos atentar quando formos fazer a manipulacao, isso pois ele sempre nos retorna um array, mesmo que possua apenas um elemento com o mesmo nome.

elementById("")

Sempre que usamos um id ele usara "element" e esta no singular, isso pois nao podemos possuir mais de uma id com mesmo nome.

elementsByClassName("")

Obs: aqui podemos observar que o "elements" volta para o plural, isso pois podemos ter mais de um elemento com a mesma class.

querySelectorAll("")

Sempre ira nos retornar um array, mesmo se tiver apenas um elemento do que selecionamos.

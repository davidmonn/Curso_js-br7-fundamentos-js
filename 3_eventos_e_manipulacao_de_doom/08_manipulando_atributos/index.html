Manipulando Atributos

Intro:

Iremos aprender como estar manipulando atributos de elementos (propriedades) .

Nessa aula iremos usar como exemplo a tag INPUT, por possuir diversos atributos para podermos estar testando.

getAttribute('')

Com o método getAttribute('') podemos verificar o valor de um atributo de uma tag HTML.

hasAttribute('')

O método hasAttribute() é usado para fazer uma verificação booleana. Ele retorna true se o atributo existir no elemento, e false se não existir. Ex de uso:

setAttribute()

Com o método setAttribute() podemos adicionar ou alterar um atributo de uma tag HTML, definindo também o valor desse atributo. Importante: se o atributo ja existir ele e alterado e se não existir é criado.

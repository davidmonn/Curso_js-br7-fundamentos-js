Selecionando Elementos 01

Intro:

Agora veremos um dos itens essenciais quando estamos tratando de manipulação de DOM.

Explicação de Manipulação DOM

Se quisermos fazer algo como, por exemplo, mudar o texto de uma tag h1 com JavaScript, podemos ter um botão onde, ao clicar, ele muda o texto para outro.

Antes de qualquer alteração, precisamos identificar o elemento que queremos modificar. Através do JavaScript, iremos navegar no DOM para encontrar esse elemento.

Na prática:

Para começar e ficar mais fácil de compreender, iremos usar o console.log, onde podemos ter uma visualização melhor.

Quando usamos document, ele é um objeto onde temos acesso a várias funções relacionadas ao documento (o conteúdo da página).

Existem dois objetos principais: window e document.

window: objeto relacionado à janela inteira do navegador, como URL, tamanho da tela, alert, etc.

document: objeto ligado ao conteúdo do site (HTML).

document.

Quando usamos document. (com ponto no final), temos acesso a funções do documento. Uma delas serve para selecionar elementos. Existem várias formas de seleção, como por tag, class, id ou combinações.

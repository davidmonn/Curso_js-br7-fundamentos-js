Manipulacao de Elementos 3

Intro:

Aqui iremos ver mais alguns itens de manipulacao de elementos, porem nao iremos mais adicionar coisas dentro de um elemento mas ao redor desse elemento, assim adicionando antes ou depois do elemento.

Explicacao

after("")

Usado para poder ser adicionado conteudo depois do elemento selecionado. Nele apenas conseguimos fazer uso de strings, nao funcionando uso de tags HTML

before("")

Usado para adicionado conteudo antes do elemento selecionado.

Como adicionar tags a after/before

Tanto para after quanto before temos duas alternativas para podermos estar adicionando tags HTML. Podemos criar elemento do zero ou podemos pegar o elemento de algum lugar e colocar onde precisamos.

Manipulacao de Elementos 3

Intro:

Aqui iremos ver mais alguns itens de manipulacao de elementos, porem nao iremos mais adicionar coisas dentro de um elemento mas ao redor desse elemento, assim adicionando antes ou depois do elemento.

Explicacao

after("")

O método after() é usado para adicionar conteúdo depois do elemento selecionado. Ele pode receber: textos(strings), elementos HTML (tags) e elementos criados no JS.

before("")

Usado para adicionado conteudo antes do elemento selecionado.

Como adicionar tags em after() e before()

Tanto o after() quanto o before() permitem adicionar tags HTML ao redor de um elemento (antes ou depois dele).

Temos duas formas principais de adicionar esses elementos:

1) Criar o elemento do zero usando JavaScript, por exemplo com document.createElement(). Usamos essa forma quando o elemento ainda não existe na página.

2) Pegar um elemento que já existe no HTML e mover ele para outra posição usando before() ou after(). Usamos essa forma quando o elemento já foi criado no HTML e só queremos reposicioná-lo.

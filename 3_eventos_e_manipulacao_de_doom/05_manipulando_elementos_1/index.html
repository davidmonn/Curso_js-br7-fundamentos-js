Manipuando Elementos 01

Intro:

Quando dizemos manipular elemento significa fazer qualquer tipo de alteracao nesse elemento, e temos diversas formas de estarmos fazendo isso.

Explicacao

Para essa aula iremos ter como exemplo um botao e nele teremos as seguintes informacoes importantes: atributo onclick para chamar a funcao.

Selecionando com children

Importante entender: Neste exemplo, teremos uma div que chama a função através do atributo onclick="testando()". Dentro dessa div existirão duas listas (ul), e cada lista conterá dois itens (li).

Quando usamos children, estamos selecionando apenas os filhos diretos (imediatos) de um elemento. Ou seja, se tivermos uma div e dentro dela existir uma ul com li, o children vai retornar apenas a ul, pois ela é o filho direto da div. Os li não são selecionados, porque são filhos da ul (filhos do filho).

Podemos fazer da seguinte forma para selecionar nosso elemento ul:

console.log(teste.children): ira selecionar nossas 2 ul. Obs: teste de nosso exemplo é a variavel que esta sendo armazenada nosso querySelector

console.log(teste.children[0].children): Agora irá mostrar os li dentro da ul. Isso acontece porque a ul também é um elemento. Ao usarmos [0], estamos acessando a primeira ul (filho da div) e, em seguida, com children, pedimos para selecionar os filhos imediatos dela, que são os li.

  • ul 1 - item 1
  • ul 1 - item 2
  • ul 2 - item 1
  • ul 2 - item 2

Exemplo children:

innerHTML

Nos retorna o conteúdo do elemento em forma de string, incluindo as tags HTML internas. É útil para verificar, ler ou alterar o conteúdo que existe dentro de um elemento.

Se quisermos usar para fazer alteracoes podemos fazer da seguinte forma:

ul.innerHTML = < li > item alterado < / li >

Exemplo innerHMTL

ou

innerText

É parecido com innerHTML, porém retorna apenas o texto visível, sem as tags HTML. Ou seja, ele ignora as tags e mostra somente o conteúdo em texto.

outerHTML

Retorna o elemento completo em forma de string, incluindo a própria tag HTML e todo o seu conteúdo interno. Ou seja, mostra o elemento inteiro, não apenas o que está dentro dele.

Se for usado para alterar o valor (assim como o innerHTML), ele não altera apenas o conteúdo, mas substitui o elemento inteiro por um novo código HTML.

Manipulando o CSS

Intro:

Aqui iremos ver como podemos modificar o CSS diretamente nos elementos usando JavaScript.

Explicação

As mudanças que iremos testar serão feitas usando CSS Inline.

Propriedade style

Uma vantagem de manipular o CSS via JavaScript é que o JS entende as propriedades de CSS.

Exemplo para modificar a cor de fundo: li.style.backgroundColor

Quando usamos style, se a propriedade não existir no elemento, ela será adicionada automaticamente.

Importante: quando a propriedade no CSS possui hífen (ex: background-color), no JavaScript usamos o padrão camelCase, ficando: backgroundColor.

Dica importante: quando queremos aplicar o mesmo estilo em vários elementos, não é uma boa prática sair aplicando style em cada um deles.

Nesses casos, o mais indicado é criar uma class no CSS e aplicar essa classe nos elementos via JavaScript.

Exemplos:

  • Primeira ul - li 1
  • Primeira ul - li 2
  • Primeira ul - li 2
  • Segunda ul - li 2

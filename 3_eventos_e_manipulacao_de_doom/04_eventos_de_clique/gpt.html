Eventos de Clique

Intro:

Evento é quando algo acontece no navegador. No JavaScript, podemos reagir a eventos do usuário, como cliques, teclado, movimento do mouse, carregamento da página, entre outros.

No caso de eventos de clique, quando clicamos em um botão, ocorrem três etapas:

  • mousedown → quando o botão do mouse é pressionado.
  • mouseup → quando o botão do mouse é solto.
  • click → quando o pressionar e soltar acontecem juntos.

Entendendo Eventos de Clique

Existem duas formas principais de executar código quando um evento de clique acontece:

  1. onclick (direto no HTML):
    • Usamos o atributo onclick dentro da tag HTML para chamar uma função do JavaScript. Essa forma é mais simples, mas não é muito recomendada em projetos grandes.

  2. Através do JavaScript (addEventListener)
    • Nesse modo, adicionamos o evento diretamente no JavaScript, sem misturar código JS no HTML. Primeiro selecionamos o elemento e depois usamos: addEventListener("click", criou)
    • O método addEventListener recebe dois parâmetros:
      • O nome do evento (ex: "click").
      • A função que será executada.
    • Também podemos criar a função diretamente dentro do parâmetro (função anônima ou arrow function).

Mais informações

Ambas as formas funcionam, porém addEventListener é mais profissional e usado em projetos reais, pois separa o HTML do JavaScript.

Resumo Rápido

  • onclick → simples, mas não recomendado em projetos grandes.
  • addEventListener → forma profissional e mais flexível.
  • click → evento mais usado para interação com o usuário.
